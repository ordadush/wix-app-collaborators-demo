# About Building Apps with AI

You can now build Wix apps by describing what you want to an AI agent. Instead of writing code from scratch, you explain your app idea to the AI agent, and it generates a complete app project using the Wix CLI. The AI agent writes the code, sets up your app dashboard, and configures extensions based on your requirements. You work in a [workspace]([https://dev.wix.com/docs/build-apps/devibe/about-app-builder-workspace](https://dev.wix.com/docs/api-preview/building-apps-with-ai/about-building-apps-with-ai#the-workspace)) that provides all the tools you need to develop and manage your app.
